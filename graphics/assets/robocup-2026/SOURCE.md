# RoboCup 2026 brand assets

Official RoboCup 2026 (Korea) event branding — symbol, English/Korean wordmarks
(horizontal + vertical), and background patterns. Used for event-themed graphics in
this package and as static brand sources in the OBS scene (`obs-template`).

- `svg/` — source of truth (scalable).
- `png/` — rasterised exports for direct use in OBS Image sources / web.

These are event identity assets for running the RoboCup 2026 SSL stream. When the
per-event theming split lands (`graphics/events/<event>/`), this folder moves to
`graphics/events/robocup-2026/assets/`.
